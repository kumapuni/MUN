# MUN
模擬国連用ソフトです

## 使い方

### 開発モード

```bash
npm install
npm run dev
```

### 本番ビルド

```bash
npm run build
npm run preview
```

### 表示画面の開き方

- 操作画面で「表示ウィンドウを開く」を押すと、別ウィンドウでプロジェクター表示が開きます。
- 表示用ウィンドウはフルスクリーンにして利用してください。

### オフライン対応

- PWA対応済みです。初回アクセス後はオフラインでも動作します。

### 資料ファイル

- PPTX/XLSXはブラウザ環境によっては表示できません。PDF形式の資料が推奨です。
