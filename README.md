### 表示画面の開き方

- 操作画面で「表示ウィンドウを開く」を押すと、別タブでプロジェクター表示が開きます。
- 表示用ウィンドウはウィンドウ化したのち、フルスクリーンにして利用してください。（Windowsの場合はほとんどのブラウザで「F11」でできます。）
